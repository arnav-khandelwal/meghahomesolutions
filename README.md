# MeghaHome

A custom web application developed as a freelance project to meet the client’s specific business requirements.

## Features

* Responsive and modern user interface
* Google Sheets lead capture integration
* GSAP animations & scroll interactions
* Optimized performance
* Mobile-friendly design

## Tech Stack

* Frontend: React / Next.js (TypeScript)
* Styling: Sass (SCSS) / CSS Modules
* Animations: GSAP / ScrollTrigger
* Integration: Google Apps Script API
* Other: Lucide React

## Notes

This project was developed as a freelance engagement. Proprietary assets, API keys, and client-specific information have been excluded from this repository.
